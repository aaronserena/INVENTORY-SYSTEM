<h1>Web-Based Inventory Management System</h1>

<p>
This is a real-time web inventory system designed to track products, manage stock, and handle sales with a minimalist interface.

It works by recording product details and sales transactions, calculating total earnings and potential revenue, and providing analytics dashboards (weekly, monthly, yearly). The built-in POS feature lets users process sales, calculate change, and print receipts. The system also supports fullscreen mode and offline use via PWA.
</p>
